I14Y API Consoles

Explore the I14Y APIs

The metadata from the I14Y interoperability platform can be obtained and maintained via the electronic interface (API). I14Y provides two APIs for this purpose.

Public API

Openly published descriptions and data concepts can be accessed via the Public API. No authentication is required. The Public API is intended for testing and occasional use. Please see the Fair Use Policy.

Swagger Handbook

Partner API

I14Y partner organisations can use the Partner API to access the organisation's internal metadata and data concepts. They can also automatically maintain their catalogues. An access token is required.
